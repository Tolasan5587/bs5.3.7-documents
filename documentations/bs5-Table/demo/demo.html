សំណួរញឹកញាប់ (FAQ)

ចុច “ចុះឈ្មោះ Vendor” បំពេញពត៌មាន ហើយក្រុមការងារយើងនឹងអនុម័តក្នុងរយៈពេលខ្លី។

Version នេះផ្តោតលើការផ្សព្វផ្សាយ និងភ្ជាប់ទំនាក់ទំនង។ ប្រព័ន្ធ Checkout អាចបន្ថែមពេលក្រោយ។

ទិន្នន័យទស្សនា និងការទំនាក់ទំនងត្រូវបានបង្ហាញក្នុង Dashboard Admin (អាចអភិវឌ្ឍបន្ថែម)।

UI អាចប្តូរភាសា (Kh/En) តាមការកំណត់។ Version Frontend នេះរៀបចំរួចសម្រាប់ bilingual។
